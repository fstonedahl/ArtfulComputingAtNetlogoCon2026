# Artful Computing Workshop @ Netlogo Conference 2026

## Workshop Part 1

For the first part of the workshop, all you need is:

1. NetLogo 7.x
2. Download and extra the [example files](art_workshop_examples.zip)
3. A creative mindset!
   
## Workshop Part 2

**BUT for the SECOND PART**, if you want to make more advanced 3D artistic renderings (with shadows and reflections), and make movies from those, you must install third party software.

### For LINUX machines:

1. **Install POV-Ray:** `sudo apt install povray`
   *(for Debian-based distros. For other distros, check your package manager)*
2. **Install OpenShot GUI video editor:** From [https://www.openshot.org/](https://www.openshot.org/)
   **OR** `ffmpeg` command line tool for video creation: `sudo apt install ffmpeg`

### For WINDOWS machines:

1. **Download POV-Ray 3.7 installer:** [https://www.povray.org/download/](https://www.povray.org/download/), run it, and also run the extra "editor" installer the first installer will prompt you to download!
2. **Install OpenShot video editor:** From [https://www.openshot.org/](https://www.openshot.org/)
   **OR** download `ffmpeg.exe` from [https://tinyurl.com/ffmpegexe](https://tinyurl.com/ffmpegexe) and put it in the same folder as the other workshop materials.

### For MAC machines:

On Mac, installing POV-Ray 3.7 requires a package manager like Homebrew or MacPorts. Homebrew is more popular in general, so let's try that.

1. If you don't have it already, install the **"Homebrew"** package manager for Mac: [https://brew.sh/](https://brew.sh/)
   *(You might have to install Xcode command line tools first: `xcode-select --install`)*
2. From the terminal, run: `brew install povray`
3. **Install OpenShot video editor:** From [https://www.openshot.org/](https://www.openshot.org/)
   **OR** `brew install ffmpeg` *(if you prefer command line video creation vs. OpenShot)*
